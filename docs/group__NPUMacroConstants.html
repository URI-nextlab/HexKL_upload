◆ HEXKL_N_COL_BYTES_MAX

#define HEXKL_N_COL_BYTES_MAX   (10240U)

Maximum number of bytes per output column supported for matrix multiplication.

This constant sets the maximum column size (in bytes) for matrix multiplication on Hexagon v73/v75/v79 and reflects constraints imposed by the 32-bit address space on these architectures.

This size is tuned to support a total memory footprint of approximately 2 GB.
